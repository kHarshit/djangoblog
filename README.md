# djangoblog

https://harshitblog.herokuapp.com/

The blog is built in Django, a high-level web framework in python.
It's hosted on Heroku.

**Features:**

* Edit and delete posts
* Drafts: Save posts for future
* User login
* User comments
* Intuitive UI

**Tools & Technologies used:**

* Django
* Python
* PostgreSQL
* HTML5
* CSS
* Bootstrap
* Heroku
